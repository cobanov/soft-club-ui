# Contributing

## Local Setup

```sh
pnpm install
pnpm dev
```

Run Storybook while developing components:

```sh
pnpm storybook
```

## Before Opening A PR

```sh
pnpm lint
pnpm typecheck
pnpm build
```

## Component Guidelines

- Use accessible primitives for interactive components.
- Prefer `forwardRef` for exported DOM-backed components.
- Support `className` overrides.
- Keep styling in `packages/ui/src/styles.css` and use tokens from `packages/tokens/src/soft-club.css`.
- Avoid hardcoded component colors, radius, spacing, shadows, and motion values outside the token layer.
- Preserve the Soft Club direction: late-90s green glass technology, Helvetica-style typography, dark surfaces, thin borders, blurred depth, and calm interaction states.

## Release Notes

The library package is `@cobanov/soft-club-ui`. Build output is generated by `tsup` as ESM, CJS, and TypeScript declarations.
